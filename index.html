Guía original

Hay que recordar que en GitHub tenemos que crear un repositorio y en la parte del nombre tiene que quedar username.github.io para que se puedan usar las paginas de GitHub y sea nuestro blog.

Instalar Node.

En primer lugar hay que instalar Hexo en tu pc, la gran ventaja sobre otros generadores de paginas es que es muy fácil hacerlo. Porque esta hecho sobre Node.js.
Es necesario ir a la pagina de Node.js y descargar la versión LTS que es la mas estable, si tu sistema es Windows solo debes descargar el archivo .exe. En caso de ser un sistema linux o Mac también están los instaladores.

Hay que asegurarse que la versión que estamos descargando tenga npm para poder instalar nuestros programas.
